FER-010: Heurísticas de Usabilidade

Ensinar as principais regras para criar interfaces intuitivas

Heurísticas 6-10

6. Reconhecimento ao invés de Recordação

"Torne objetos, ações e opções visíveis. O usuário não deve ter que lembrar informações de uma parte do diálogo para outra."

  • Mostre opções visíveis ao invés de escondidas
  • Use labels e hints claros
  • Evite que usuários precisem memorizar

7. Flexibilidade e Eficiência

"Atalhos - invisíveis para usuários novatos - podem acelerar a interação para usuários experientes."

  • Forneça atalhos para usuários experientes
  • Permita personalização
  • Mantenha caminhos simples para iniciantes

8. Design Estético e Minimalista

"Diálogos não devem conter informações irrelevantes ou raramente necessárias."

  • Remova elementos desnecessários
  • Foque no essencial
  • Evite poluição visual

9. Ajudar Usuários a Reconhecer, Diagnosticar e Recuperar de Erros

"Mensagens de erro devem ser expressas em linguagem simples, indicar precisamente o problema e sugerir uma solução construtiva."

  • Mensagens claras e acionáveis
  • Explique o problema e como resolver
  • Use linguagem simples

10. Ajuda e Documentação

"Mesmo que seja melhor se o sistema puder ser usado sem documentação, pode ser necessário fornecer ajuda e documentação."

  • Documentação fácil de encontrar
  • Foque em tarefas do usuário
  • Mantenha ajuda atualizada
