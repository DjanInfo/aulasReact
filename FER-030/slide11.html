FER-029: Fundamentos do React e Virtual DOM

Compreender a arquitetura e funcionamento interno do React

Exercícios aplicados

1️⃣ Painel de indicadores

Crie um componente Dashboard que recebe via props um array de indicadores {'{ label, value }'} e renderiza cards.

  • Permita inserir um botão “ver tudo” usando props.children
  • Destaque valores acima de um limite também recebido por props

2️⃣ Formulário controlado

Implemente ProfileForm recebendo um objeto user e um callback onSubmit.

function ProfileForm({ user, onSubmit }) { const [form, setForm] = useState(user); // ... }

Quando user mudar, sincronize o estado local com useEffect.

3️⃣ Controle de tarefas

Faça um componente TodoList que:

  • Recebe todos e onToggle(id) via props
  • Possui estado local apenas para o filtro (all, done, pending)
  • Deriva a lista filtrada dinamicamente sem criar outro useState

4️⃣ Exercício bônus

Transforme uma seção repetitiva em componente composto (Section.Header, Section.Body, Section.Footer) expondo slots nomeados.

Dicas

  • Desenhe o fluxo de dados antes de codar
  • Valide props com TypeScript ou PropTypes
  • Teste callbacks com console.log para garantir a direção correta dos dados
